# ⚡ Invincible Energy | Intelligence Agent 

An automated, AI-driven market intelligence dashboard for **Invincible Energy**. This system monitors Australian energy markets (STC, LGC, Wholesale) and grid stability in real-time.

## 🚀 System Architecture
- **Brain (`scraper.py`)**: A Python-based engine that generates market intelligence data.
- **Automation (`sync.yml`)**: GitHub Actions workflow that triggers the brain every 10 minutes.
- **Frontend (`index.html`)**: A high-performance, dark-mode dashboard with real-time status indicators.
- **Deep Dives**: 4 dedicated intelligence layers for STC, LGC, Wholesale, and Grid Ops.

## 🛠️ Tech Stack
- **Languages**: Python 3.x, HTML5, CSS3, JavaScript (ES6+).
- **Hosting**: GitHub Pages.
- **Automation**: GitHub Actions (CI/CD).
- **Data Format**: Dynamic JSON streaming.

## 📈 Key Features
- **Real-Time Clock**: Syncs with local AEST time.
- **Risk Indicators**: Dynamic risk bars (0-100%) based on market volatility.
- **Status Ticks**: Green (✓) for stable/upward trends, Red (✕) for high-risk/downward trends.
- **Mobile Responsive**: Fully optimized for iPhone/Android field use.
